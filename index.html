Project Description

About This Project

This website was built by Group 16 as part of the Cloud Computing Practical Lab 1 pre-lab assignment at the university. Our group of four students worked together to design and deploy a multi-page static website using HTML, CSS, and JavaScript.

The site is hosted on GitHub Pages — giving us real, hands-on experience with cloud-based web deployment, version control using Git, and team collaboration through a shared GitHub repository.

As part of Part 1 of the lab, every group member also installed Oracle VM VirtualBox on their personal laptops to understand the fundamentals of virtualization — the core technology that powers modern cloud computing infrastructure.
