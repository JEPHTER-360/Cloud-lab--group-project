Our Project

What This Project Is About

This website was created as part of Cloud Computing Practical Lab 1 at the university. Our group collaborated to design and deploy a static website using HTML, CSS, and JavaScript — hosted via GitHub Pages as part of our introduction to cloud-based deployment.

The project demonstrates our ability to work as a team, manage a shared GitHub repository, and apply fundamental web development skills in a cloud context.
