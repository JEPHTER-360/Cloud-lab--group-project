Project Overview

About This Project

For Cloud Computing Lab 1, Group 16 collaborated to design and deploy a multi-page static website using HTML, CSS, and JavaScript. The site is hosted on GitHub Pages, giving us hands-on experience with cloud-based deployment, version control, and team-based web development.

We also installed Oracle VM VirtualBox on our personal laptops to explore the fundamentals of virtualization — a core technology behind modern cloud computing infrastructure.
