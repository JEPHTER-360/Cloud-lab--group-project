Group 16 — Cloud Computing

We are Group 16, a team of four university students enrolled in the Cloud Computing course. For Practical Lab 1, our group collaborated to build and deploy a fully functional static website using HTML, CSS, and JavaScript, hosted on GitHub Pages.

Every member of our group contributed to the project — from writing HTML and CSS to uploading files, making commits, and helping manage the shared GitHub repository. We also each completed the individual task of installing Oracle VM VirtualBox to gain hands-on experience with virtualization technology.
